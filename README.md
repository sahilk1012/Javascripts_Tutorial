# JavaScript Concepts & Reference Files 🟨

Welcome! This repository is a collection of standalone JavaScript files, code snippets, and notes. It is designed as a quick reference guide to help anyone understand how different JavaScript concepts work. 

There are no large applications or complex setups here—just clean, isolated examples of JavaScript in action.

## 📂 What's Inside

The repository is organized by topic. Inside the files, you will find code examples along with comments explaining the syntax and logic. 

*   **Core Basics:** Variables (`let`, `const`, `var`), Data Types, and Operators
*   **Control Flow:** Loops (`for`, `while`), If/Else statements, Switch cases
*   **Functions:** Arrow functions, Callbacks, Higher-Order Functions
*   **Data Structures:** Array methods (`map`, `filter`, `reduce`), Objects, JSON
*   **Modern JavaScript (ES6+):** Destructuring, Spread/Rest operators, Template Literals
*   **Asynchronous JS:** Promises, Async/Await, Timeouts\

*   ## 🛠️ How to Use This Repository

Since this isn't a single connected project, you can jump straight into whatever file or topic interests you. You can simply read the files directly on GitHub to get an idea of how the code works, or test them out yourself.
